Never Out of Shape!

Responsive Design

September 8, 2022 Emmanuel Oladosu

In the early days of web design, pages were built to target a particular screen size. If the user had a larger or smaller screen than the designer expected, results ranged from unwanted scrollbars to overly long line lengths, and poor use of space. Responsive Web Design coined by Ethan Marcotte in 2010 is a graphic user interface (GUI) design approach used to create content that adjusts smoothly to various screen sizes, making a website look good on all devices using HTML and CSS to automatically resize a website.
